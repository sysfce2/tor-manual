প্রথমবার Tor Browser চালাচ্ছি

When you run Tor Browser for the first time, you will see the Connect to Tor window. This offers you the option to either connect directly to the Tor network, or to configure Tor Browser for your connection. There's a checkbox which asks whether you always want to get automatically connected to the Tor network, if this is the case, check the box.

সংযোগ

বেশিরভাগ ক্ষেত্রে, "Connect" নির্বাচন করা আপনাকে কোনও আর কনফিগারেশন ছাড়াই Tor network সাথে সংযোগ করার অনুমতি দেবে।

Once clicked, a status bar will appear, showing Tor's connection progress. আপনি যদি তুলনামূলকভাবে দ্রুত সংযোগে থাকেন তবে এই বারটি একটি নির্দিষ্ট সময়ে আটকা পড়েছে বলে মনে হচ্ছে, সমস্যা সমাধানে সহায়তার জন্য Troubleshooting পৃষ্ঠাটি দেখুন।   Or, if you know that your connection is censored or uses a proxy, you should click on "Tor Network Settings".

কনফিগার

Tor Browser will take you through a series of configuration options.

The first checkbox is Quickstart. If selected, every time you open Tor Browser, it will try to connect with your previous network settings.

The second checkbox asks if you want to use a bridge. If you know your connection is censored, or you have tried and failed to connect to the Tor network and no other solutions have worked, select "Use a bridge".

This will display the Circumvention section to configure a pluggable transport or to connect using Bridges.

OTHER OPTIONS

The third checkbox asks if your connection uses a proxy. In most cases, this is not necessary. You will usually know if you need to select this checkbox because the same settings will be used for other browsers on your system. If possible, ask your network administrator for guidance. If your connection does not use a proxy, click "Connect".
