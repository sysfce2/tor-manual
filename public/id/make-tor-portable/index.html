MEMBUAT TOR BROWSER PORTABEL

Jika diinginkan, Tor Browser dapat dibuat portabel dengan mengekstraknya dari arsipnya langsung ke media yang dapat dipindahkan seperti stik USB atau kartu SD. Disarankan untuk menggunakan media yang writable agar Tor Browser dapat diperbarui sesuai kebutuhan.

Untuk Windows:

  1. Colokkan media yang dapat dipindahkan dan format. Semua jenis sistem file akan berfungsi.

  2. Arahkan ke Tor Browser download page.

  3. Unduh berkas Windows .exe dan simpan langsung ke media Anda.

  4. (Disarankan) periksa Berkas signature.

  5. Setelah pengunduhan selesai, klik berkas .exe dan mulai proses instalasi.

  6. Saat penginstal bertanya di mana menginstal Tor Browser, pilih media yang dapat dipindahkan.

Untuk macOS:

  1. Colokkan media yang dapat dipindahkan dan format. Anda harus menggunakan format Extended macOS (Journaled).

  2. Arahkan ke Tor Browser download page.

  3. Unduh berkas macOS .dmg dan simpan langsung ke media Anda.

  4. (Disarankan) periksa Berkas signature.

  5. Setelah pengunduhan selesai, klik file .dmg dan mulai proses instalasi.

  6. Saat penginstal bertanya di mana menginstal Tor Browser, pilih media yang dapat dipindahkan.

Untuk GNU/Linux:

  1. Colokkan media yang dapat dipindahkan dan format. Semua jenis sistem file akan berfungsi.

  2. Arahkan ke Tor Browser download page.

  3. Download berkas Linux .tar.xz dan simpan langsung ke media Anda.

  4. (Disarankan) periksa Berkas signature.

  5. Saat unduhan selesai, ekstrak arsip ke media juga.
